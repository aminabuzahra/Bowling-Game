##  Simple Bowling Video Game
### Unity version: 2019.4.17f1

1. Add the project to Unity Hub
2. Make sure the above mentioned Unity version (or later) is selected 
3. Open the project
4. Open BowlingScene from :
```
	├── Assets
	│   └── Scenes
	|       |-- BowlingScene.unity
```
5. Save the project.
